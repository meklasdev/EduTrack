Funkcji gotowych
Funkcjonalności

Wszystko, czego potrzebuje
nowoczesna sala egzaminacyjna

📊
MultiGrader — 8 formatów
Excel, Word, PowerPoint, SQL, Packet Tracer, SVG/GIMP, Wireshark pcap oraz kod C++/Python. Partial credit, normalizacja i porównanie z wzorcem.
🤖
AI Engine (Ollama)
Lokalny LLM bez danych w chmurze. Ocenianie kodu (styl + logika), generowanie egzaminów PL na słabe strony, narracja wyników i detekcja zagrożeń.
🛡️
Anti-Cheat Engine
OCR zrzutów ekranu (tesseract), analiza zachowania myszy, wykrywanie USB, blokada sieci iptables/netsh, auto black-screen po 10 alertach.
🖥️
Panel Nauczyciela
Mapa klasy 3D, heatmapa aktywności 24h, ranking z medalami 🥇🥈🥉, marketplace szablonów, 4 skórki kolorystyczne, 8 skrótów klawiaturowych.
🎓
Terminal Ucznia
Wbudowany jspreadsheet CE v4, timer egzaminacyjny, asystent dokumentacji (36 wpisów), tryb web i Electron, tryb offline z lokalnym cache.
🔐
Bezpieczeństwo & Auth
JWT + bcrypt, izolacja danych per dział, Redis-based sessions, rate limiting 60 req/min, TLS WebSocket, skalowanie klastrowe (CLUSTER=true).
📄
Raporty & Certyfikaty
PDF cheat evidence z listą dowodów, certyfikaty ukończenia, detekcja plagiatów (≥90% = alert), JSON exports, wspólna ocena multi-nauczycielska.
🌐
Ekosystem
Chrome Extension MV3 dla Chromebook, system wtyczek 3rd-party (server/plugins/), Docker Compose, backup pg_dump + S3, auto-installer Ubuntu.
⚙️
DevOps & Infra
Logowanie ELK (Winston JSON), zaplanowany backup S3, Raspberry Pi arm64 support, Node.js cluster mode, Headless CLI mode, mDNS auto-discovery.

Prosty przepływ,
potężne możliwości

1
Nauczyciel loguje się
JWT auth, wybiera dział, wgrywa szablon wzorcowy do AutoGradera.
2
Uczniowie dołączają
Electron agent lub przeglądarka. Auto-discovery przez mDNS w LAN.
3
Start zadania
Nauczyciel wysyła task push. Timer startuje, arkusz się odblokowuje.
4
Monitoring RT
Screenshoty co 3s, OCR w tle, analiza myszy, alerty w czasie rzeczywistym.
5
Sprawdzenie
Student wysyła plik, MultiGrader + AI porównuje z wzorcem w sekundy.
6
Wyniki & Ranking
Leaderboard na żywo, certyfikaty PDF, raporty cheat evidence.

    Instalacja

    Uruchom w 5 minut
    na Ubuntu

    🖥️ Serwer (Ubuntu 20.04+)
    # 1. Klonuj i zainstaluj git clone <repo-url> && cd EduTrack chmod +x setup-ubuntu.sh ./setup-ubuntu.sh # 2. Uruchom bazy danych docker-compose up -d # 3. Konfiguracja cd server && cp .env.example .env # → Ustaw DATABASE_URL, JWT_SECRET # 4. Migracja i start npx prisma migrate deploy npm start # Panel: http://localhost:8080/login
    🎓 Terminal ucznia (Electron)
    # 1. Wejdź do katalogu cd app-student # 2. Zainstaluj zależności npm install # 3. Uruchom agenta npm start # ✅ Agent automatycznie wykrywa # serwer w sieci LAN przez mDNS. # Przeglądarka: # http://<server-ip>:8080
