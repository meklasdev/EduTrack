Funkcjonalności

Wszystko, czego potrzebuje
nowoczesna sala egzaminacyjna

📊
AutoGrader Excel
Silnik ExcelLogicChecker porównuje formuły, wartości i formatowanie. Obsługa VLOOKUP, SUMA.JEŻELI, formuł zagnieżdżonych i tolerancji zaokrąglenia ±0.01.
🛡️
Anti-Cheat Engine
Wykrywanie AI (ChatGPT, Claude, Gemini), monitoring okien przez Windows API, auto black-screen po 10 alertach, detekcja utraty focusu.
🖥️
Panel Nauczyciela
Dashboard w stylu MS Teams z widokiem w czasie rzeczywistym. Miniatury ekranów, ranking, mapa klasy 3D, toast notifications, skróty klawiszowe.
🎓
Terminal Ucznia
Wbudowany arkusz kalkulacyjny jspreadsheet CE v4, timer egzaminacyjny, asystent dokumentacji (36 wpisów), tryb web i Electron.
🔐
Multi-Teacher Auth
JWT + bcrypt, izolacja danych per dział, przypisywanie uczniów do nauczyciela, Redis-based session management, rate limiting.
📄
Raporty & Certyfikaty
Generowanie PDF raportów cheatowania, certyfikatów ukończenia, detekcja plagiatów (porównanie formuł 90%+ = plagiat), JSON exports.
Przebieg egzaminu

Prosty przepływ,
potężne możliwości

1
Nauczyciel loguje się
JWT auth, wybiera dział, wgrywa szablon .xlsx do AutoGradera.
2
Uczniowie dołączają
Electron agent lub przeglądarka. Auto-discovery przez mDNS w sieci LAN.
3
Start zadania
Nauczyciel wysyła task push. Timer startuje, arkusz się odblokowuje.
4
Monitoring RT
Screenshoty co 3s, analiza okien co 5s, alerty w czasie rzeczywistym.
5
Sprawdzenie
Student wysyła plik, ExcelChecker porównuje z wzorcem, wynik w sekundy.
6
Wyniki & Ranking
Leaderboard na żywo, certyfikaty PDF, raporty cheat evidence.

    Instalacja

    Uruchom w 5 minut
    na Ubuntu

    🖥️ Serwer (Ubuntu 20.04+)
    # 1. Klonuj i zainstaluj git clone <repo-url> && cd EduTrack chmod +x setup-ubuntu.sh ./setup-ubuntu.sh # 2. Uruchom bazy danych docker-compose up -d # 3. Konfiguracja cd server && cp .env.example .env # → Ustaw DATABASE_URL, JWT_SECRET # 4. Migracja i start npx prisma migrate deploy npm start # Panel: http://localhost:8080/login
    🎓 Terminal ucznia (Electron)
    # 1. Wejdź do katalogu cd app-student # 2. Zainstaluj zależności npm install # 3. Uruchom agenta npm start # ✅ Agent automatycznie wykrywa # serwer w sieci LAN przez mDNS. # Działa również jako strona web: # http://<server-ip>:8080
    ✅ Wymagania:
    Node.js 20+  ·  Docker  ·  Ubuntu 20.04+
