# EduTrack Pro v1.0

Kompleksowy system zarządzania salą informatyczną i automatycznego oceniania zadań.

## 🚀 Funkcje
- **Terminal Ucznia:** Aplikacja Electron z wbudowanym arkuszem kalkulacyjnym.
- **Automatyczne Ocenianie:** Sprawdzanie formuł, wartości i stylów w Excelu.
- **Anti-Cheat:** Wykrywanie AI (ChatGPT, Claude, itp.), monitorowanie Alt-Tab i blokada komunikatorów.
- **Panel Nauczyciela:** Podgląd ekranów live, timeline aktywnych okien, zdalna blokada komputera.
- **Asystent Dokumentacji:** Wbudowana pomoc dla INF.02, INF.03, Excel i C++.

## 🛠️ Instalacja (Ubuntu Server)
```bash
chmod +x setup-ubuntu.sh
./setup-ubuntu.sh
```
