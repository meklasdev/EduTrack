# 🎓 EduTrack System Requirements
### Vocational Exams: INF.02, INF.03, INF.04 (Academic Year 2025/2026)

---

## 🛠️ 1. Office & Productivity Suite
* **Spreadsheets:**
    * **Microsoft Excel** (`.xlsx`) – Primary exam standard. The EduTrack `exceljs` engine performs deep formula validation.
    * **LibreOffice Calc** (`.ods`).
* **Word Processing:** Microsoft Word, LibreOffice Writer.
* **Presentations:** PowerPoint, LibreOffice Impress.

## 💻 2. Development Environments (Coding & Dev)
* **C++:** Code::Blocks, Dev-C++, VS Code (with C++ extensions).
* **Python:** PyCharm Community Edition, IDLE.
* **Web Development:** Visual Studio Code, Sublime Text, Notepad++.
* **Database Management:** MariaDB / MySQL, HeidiSQL, DBeaver.

## 🎨 3. Graphics & Multimedia
* **Raster Graphics:** GIMP.
* **Vector Graphics:** Inkscape.
* **Utilities:** IrfanView.

## 🌐 4. Networking & Systems (INF.02 Focus)
* **Simulation:** Cisco Packet Tracer.
* **Virtualization:** Oracle VirtualBox.
* **Analysis & Diagnostics:** Wireshark, Nmap.
* **Remote Access:** PuTTY, WinSCP.

## 📦 5. System Tools & Utilities
* **Archiving:** 7-Zip.
* **Local Servers:** XAMPP / WAMP.
* **Web Browsers:** Chrome, Firefox, Microsoft Edge.

---

## 🛡️ EduTrack Agent: Functional Matrix

| Feature / Tool | EduTrack System Action | Objective |
| :--- | :--- | :--- |
| **Excel** | **Auto-Grade Parser** | Automated validation of complex formulas (e.g., `VLOOKUP`). |
| **C++ / Python** | **Sandboxed Execution** | Remote code execution and output verification in an isolated environment. |
| **GIMP / Inkscape** | **Activity Monitoring** | Integrity check to prevent the use of pre-made assets/templates. |
| **Packet Tracer** | **Automated Collection** | Seamless background retrieval of `.pkt` topology files. |
| **Browsers** | **Dynamic Whitelisting** | Restricting access to authorized documentation only. |
